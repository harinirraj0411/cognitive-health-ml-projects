# Cognitive Health ML Projects

This repository contains machine learning and data analysis projects focused on cognitive health and early detection of cognitive decline.

## Projects

### 1. Machine Learning-Based Prediction of Cognitive Decline
- Built Random Forest and Logistic Regression models
- Achieved 85%+ accuracy
- Identified key predictors such as memory score and physical activity

### 2. Exploratory Data Analysis of Cognitive Health Data
- Performed data visualization and correlation analysis
- Identified important health and lifestyle factors

### 3. Anomaly Detection for High-Risk Cognitive Cases
- Used Isolation Forest for anomaly detection
- Identified ~10% high-risk patterns in data

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
